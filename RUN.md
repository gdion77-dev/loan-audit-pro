# Loan Audit PRO — Τοπική Εκτέλεση & Προεπισκόπηση

Τεχνικός Έλεγχος Κανονικών Δανείων. Το παρακάτω εξηγεί πώς τρέχει η εφαρμογή
τοπικά και πώς ακολουθείτε τη ροή έως τη λήψη PDF.

## Προαπαιτούμενα
- Node.js 20+ και npm
- Σύνδεση δικτύου για το πρώτο `npm install` (κατεβάζει Vite/React)

## Εντολές

```bash
# 1) εγκατάσταση εξαρτήσεων (μία φορά, χρειάζεται δίκτυο)
npm install

# 2) εκκίνηση εφαρμογής (dev server με hot reload)
npm run dev
```

Η εφαρμογή ανοίγει αυτόματα στο πρόγραμμα περιήγησης στη διεύθυνση:

```
http://localhost:5173/
```

(Αν δεν ανοίξει αυτόματα, επικολλήστε τη διεύθυνση στον browser.)

### Άλλες εντολές
```bash
npm run lint     # έλεγχος τύπων (tsc --noEmit)
npm test         # σουίτα δοκιμών (node:test μέσω tsx)
npm run build    # μεταγλώττιση σε dist/
npm run preview  # προεπισκόπηση του build
```

## Ροή χρήσης μέσα στην εφαρμογή
1. **Στοιχεία Υπόθεσης / Όροι Δανείου / Επιτόκιο / Ρυθμίσεις** — συμπληρώστε τα πεδία.
2. **Δοσολόγιο Τράπεζας / Fund** — επιλέξτε «Σύμβαση ημερομέτρησης» και προσθέστε γραμμές.
3. Παρακολουθήστε τον **Φάκελο Ελέγχου** (δεξιά): η κατάσταση προσχεδίου γίνεται
   **«Έτοιμο»** όταν συμπληρωθούν όλα τα κρίσιμα πεδία.
4. **Μελέτη / PDF**: όταν είναι «Έτοιμο», πατήστε **«Εκτέλεση Μελέτης»**.
5. Μετά την εκτέλεση εμφανίζονται: κατάσταση, **«PDF: Διαθέσιμο»**, προεπισκόπηση
   κειμένου, και το κουμπί **«Λήψη PDF»**.
6. Πατήστε **«Λήψη PDF»** για λήψη του αρχείου στο πρόγραμμα περιήγησης.

> Σημείωση: Η εφαρμογή είναι αμιγώς client-side. Δεν υπάρχει backend, αποθήκευση,
> σύνδεση χρήστη, εισαγωγή Excel ή OCR. Τα δεδομένα ζουν μόνο στη συνεδρία του browser.
